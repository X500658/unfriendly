# Lesson 21 - I think

## Sentence Pattern 1 - Conjecture/Guess/Opinion
```
(Verb,普通形・Noun・Adj)と　思います
```

Don't forget to add `だ` when using Nouns and なAdj

## Sentence Pattern 2 - に　ついて (Concerning/About)
```
(Noun)に　ついて　どうおもいますか。
```
